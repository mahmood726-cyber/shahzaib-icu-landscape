      Methods

      Search strategy, classification, and known limitations.

      Search strategy

      Trials are retrieved from ClinicalTrials.gov (CT.gov) via the API v2 using free-text query.term queries filtered by AREA[StudyType]INTERVENTIONAL AND AREA[DesignAllocation]RANDOMIZED. ICU/critical care terms include: intensive care, critical care, ICU, ARDS, ECMO, septic shock, vasopressor, mechanical ventilation, respiratory failure, cardiogenic shock, hemorrhagic shock, organ dysfunction, CCU, MICU, SICU, PICU, NICU, and related synonyms. No date range restriction is applied. Page size: 1000 records. No MeSH structured queries (CT.gov API v2 does not support AREA[ConditionSearch] with MeSH). The search date is recorded in the summary JSON for PRISMA-S compliance.

      Population scope

      The search includes adult, pediatric (PICU), and neonatal (NICU) intensive care populations. Hemodynamic physiology differs across age groups; users should filter by condition text if age-specific analyses are needed. No age-based exclusion criteria are applied.

      Hemodynamic outcome classification

      Outcome measures, descriptions, and time frames are scanned for hemodynamic keywords using word-boundary matching. Keywords are normalized to 24 canonical categories (e.g., "norepinephrine" → Vasopressor/Inotrope, "MAP" → Mean Arterial Pressure). Ambiguous abbreviations (HR, SV, PAP) use negative-context filtering to exclude non-hemodynamic uses (e.g., "HR" as Hazard Ratio, "PAP" as Pap smear). Non-hemodynamic adjuncts (ICU severity scores, ventilation duration, resuscitation endpoints) are included for clinical relevance but are reported separately from core hemodynamic totals to avoid inflating evidence counts. Classification accuracy (sensitivity/specificity) has not been formally validated against a manual screening gold standard.

      Comparator arm classification

      "Placebo/sham" refers to true inert comparators only. Usual care and standard care are classified separately as active comparators. The structured CT.gov armGroupType field is used when available; text-based classification is the fallback.

      Study status handling

      All study statuses are included (recruiting, completed, withdrawn, terminated, suspended). Withdrawn trials (never enrolled) and terminated trials (stopped early) may have registered outcomes but never collected data. Status stratification is provided in the summary JSON. Users should consider filtering by status for quantitative analyses.

      Limitations

      • ClinicalTrials.gov only — trials registered solely on other registries (EUCTR, ISRCTN, ANZCTR, ChiCTR) are not captured.
      • Keyword matching operates on outcome text — precision depends on outcome description quality. Sensitivity/specificity not formally assessed.
      • ICU severity scores (SOFA, APACHE) are composite indices; only the cardiovascular subscore is hemodynamic. Reported as adjuncts.
      • Ventilator-free days and mechanical ventilation duration are respiratory outcomes, included as adjuncts for clinical context.
      • Counts are mention-level by default (one study with 3 MAP time points = 3 mentions). Toggle to study-level in the signal map.
      • Geographic bias: CT.gov over-represents US-centric and industry-sponsored trials.
      • No risk of bias / quality appraisal is performed (masking and allocation data are extracted but not synthesized).
      • This evidence gap map is not prospectively registered (PROSPERO/OSF).
      • Adapted PRISMA flow — not a full PRISMA 2020 flow diagram.
