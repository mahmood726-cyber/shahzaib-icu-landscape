      Methods

      Search strategy, classification, and known limitations.

      Search strategy

      Trials are retrieved from ClinicalTrials.gov (CT.gov) via the API v2 using free-text queries for ICU/critical care terms (intensive care, critical care, ICU, ARDS, ECMO, septic shock, vasopressor, mechanical ventilation, and related synonyms). Only interventional, randomized studies are included.

      Hemodynamic outcome classification

      Outcome measures, descriptions, and time frames are scanned for hemodynamic keywords using word-boundary matching. Keywords are normalized to canonical categories (e.g., "norepinephrine" → Vasopressor/Inotrope, "MAP" → Mean Arterial Pressure). Ambiguous abbreviations (HR, SV) use negative-context filtering to exclude non-hemodynamic uses (e.g., "HR" as Hazard Ratio is excluded when adjacent to statistical patterns).

      Comparator arm classification

      "Placebo/sham" refers to true inert comparators only. Usual care and standard care are classified separately as active comparators. The structured CT.gov armGroupType field is used when available; text-based classification is the fallback.

      Limitations

      • ClinicalTrials.gov only — trials registered solely on other registries are not captured.
      • All study statuses are included (recruiting, completed, withdrawn, terminated).
      • Keyword matching operates on outcome text — precision depends on outcome description quality.
      • ICU severity scores (SOFA, APACHE) are reported separately as they are composite indices, not pure hemodynamic parameters.
      • Counts are mention-level by default (one study with 3 MAP time points = 3 mentions).
      • Geographic bias: CT.gov over-represents US-centric and industry-sponsored trials.
