      Methods

      Search strategy, classification, and known limitations.

      Rationale and objectives

      This living trial landscape map aims to characterize the scope, coverage, and temporal trends of registered ICU randomized trials with hemodynamic and related outcomes. The objective is descriptive: to identify which hemodynamic parameters are well-represented in trial registrations and where placebo-controlled evidence is sparse. This is not a systematic review and does not synthesize treatment effects.

      Eligibility criteria

      Inclusion: Interventional, randomized studies registered on ClinicalTrials.gov with at least one ICU/critical care population term and at least one hemodynamic keyword in registered outcome measures.
      Exclusion: Observational studies, non-randomized designs, and trials without any hemodynamic keyword match are excluded from the hemodynamic subset. No date, language, or geographic restrictions.

      Information sources

      Primary: ClinicalTrials.gov API v2 (all study statuses, no date restriction).
      Supplementary: PubMed E-utilities (esearch + efetch) for published RCTs that may lack CT.gov registration. PubMed results are deduplicated against CT.gov via NCT ID cross-referencing and Levenshtein fuzzy title matching (threshold ≥ 0.85, with Jaccard word-overlap pre-filter ≥ 0.50). The 0.85 threshold was chosen to balance sensitivity against false merges for English-language clinical trial titles, which typically share standardized phrasing. A validation sample of 100 pairs (50 merged, 50 near-miss unmerged) is provided for independent assessment.
      Enrichment sources (7 adapters): PubMed (abstracts, MeSH terms), OpenFDA/FAERS (safety signals), OpenAlex (citation counts, journal metadata, author affiliations), Crossref (DOI metadata), Europe PMC (related articles), Unpaywall (open access status), OpenCitations (citation links).

      CT.gov search strategy

      Trials are retrieved via the API v2 using free-text query.term queries filtered by AREA[StudyType]INTERVENTIONAL AND AREA[DesignAllocation]RANDOMIZED. ICU/critical care terms include: intensive care, critical care, ICU, ARDS, ECMO, septic shock, vasopressor, mechanical ventilation, respiratory failure, cardiogenic shock, hemorrhagic shock, organ dysfunction, TBI, cardiac arrest, targeted temperature management, therapeutic hypothermia, major burns, pulmonary embolism, DKA, acute liver failure, status epilepticus, CCU, MICU, SICU, PICU, NICU, and related synonyms. Page size: 1000 records. No MeSH structured queries (CT.gov API v2 does not support AREA[ConditionSearch] with MeSH). The initial search was executed on 14 February 2026. Search dates are recorded programmatically in the summary JSON for each update cycle. Full query available in ctgov_icu_placebo_strategy.json.

      Recall validation: A reference standard of 21 landmark ICU RCTs spanning septic shock, ARDS, ECMO, cardiac arrest/TTM, TBI, pulmonary embolism, burns, cardiogenic shock, hemorrhagic shock, and DKA was assembled from published systematic reviews. All 21 trials (21/21 = 100% recall) were captured by the current search strategy. NCT IDs and coverage notes are documented in the configuration file. The reference standard does not yet include hepatorenal syndrome or acute pancreatitis sub-populations.

      PubMed search strategy

      PubMed is searched using E-utilities with ICU terms harmonized to the CT.gov query, combined with hemodynamic outcome terms, filtered to Randomized Controlled Trial[pt]. Results are merged with the CT.gov dataset; studies with matching NCT IDs are deduplicated. PubMed-only studies are flagged with INFERRED_COMPLETED status and INFERRED_RANDOMIZED allocation (not independently verified). PubMed introduces a temporal bias toward published (completed) trials and may under-represent recently registered, not-yet-published studies.

      Population scope

      The search includes adult, pediatric (PICU), and neonatal (NICU) intensive care populations. Hemodynamic physiology differs across age groups; users should filter by condition text if age-specific analyses are needed. No age-based exclusion criteria are applied.

      Hemodynamic outcome classification

      Outcome measures, descriptions, and time frames are scanned for hemodynamic keywords using word-boundary matching. Keywords are normalized to 24 canonical categories (e.g., "norepinephrine" → Vasopressor/Inotrope, "MAP" → Mean Arterial Pressure). Ambiguous abbreviations (HR, SV, PAP) use negative-context filtering to exclude non-hemodynamic uses (e.g., "HR" as Hazard Ratio, "PAP" as Pap smear). Non-hemodynamic adjuncts (ICU severity scores, ventilation duration, resuscitation endpoints) are included for clinical relevance but are reported separately from core hemodynamic totals to avoid inflating evidence counts. Classification accuracy was assessed using a stratified random sample (n = 200 mentions across all keyword categories). The validation framework and annotated sample are provided in validate_classification.py and validation/validation_sample_broad_200.csv. Formal sensitivity/specificity reporting requires independent human scoring of this sample, which is provided as supplementary material for reviewers.

      Data charting and items

      Extracted data items per trial: NCT ID, title, conditions, interventions, study status, phase, enrollment, start date, completion date, allocation, masking, arm descriptions, primary/secondary/other outcome measures with time frames, countries, and comparator type. Charting is fully automated from structured API fields; no manual data extraction is performed. Data dictionary available in the pipeline source code (build_living_map.py).

      Comparator arm classification

      "Placebo/sham" refers to true inert comparators only. Usual care and standard care are classified separately as active comparators. The structured CT.gov armGroupType field is used when available; text-based classification is the fallback.

      Study status handling

      All study statuses are included (recruiting, completed, withdrawn, terminated, suspended). Withdrawn trials (never enrolled) and terminated trials (stopped early) may have registered outcomes but never collected data. Status stratification is provided in the summary JSON. Users should consider filtering by status for quantitative analyses.

      Living review specification

      This trial landscape map follows a living review model with automated updates. Update frequency: Monthly incremental updates via living_update.py, executed manually by the corresponding author. The script fetches new/updated trials from CT.gov and PubMed since the last successful run. Automation via cron/CI is planned but not yet operational. Stopping criteria: Updates will continue as long as the CT.gov API remains available and the dashboard is actively maintained. Cessation will be documented with a final update date and reason. Amendment policy: Changes to the search strategy, classification rules, or enrichment sources are versioned in ctgov_icu_placebo_strategy.json and documented in the pipeline commit history. Major amendments (e.g., adding new population terms) trigger a full rebuild via --force-full. Version history: Each update cycle is logged in living_log.jsonl with timestamp, source counts, and TruthCert provenance hash.

      Limitations

      • ClinicalTrials.gov only — trials registered solely on other registries (EUCTR, ISRCTN, ANZCTR, ChiCTR) are not captured.
      • Keyword matching operates on outcome text — precision depends on outcome description quality. A stratified validation sample (n = 200) is provided for independent scoring.
      • ICU severity scores (SOFA, APACHE) are composite indices; only the cardiovascular subscore is hemodynamic. Reported as adjuncts.
      • Ventilator-free days and mechanical ventilation duration are respiratory outcomes, included as adjuncts for clinical context.
      • Counts are mention-level by default (one study with 3 MAP time points = 3 mentions). Toggle to study-level in the signal map.
      • Geographic bias: CT.gov over-represents US-centric and industry-sponsored trials.
      • Consistent with scoping review methodology (Tricco et al., 2018; JBI Manual for Evidence Synthesis), no risk of bias or quality appraisal was performed. The objective is to map the extent and nature of available evidence, not to synthesize effect estimates. Masking and allocation data are extracted for descriptive purposes but not appraised.
      • This trial landscape map was not prospectively registered. A full protocol describing the search strategy, classification rules, enrichment pipeline, and living update specification is provided as supplementary material. Protocol registration on OSF Registries will be completed before the living update cycle begins.
      • Adapted PRISMA flow — not a full PRISMA 2020 flow diagram. A completed PRISMA-ScR checklist is provided as supplementary material (Table S1: PRISMA-ScR_Checklist.md).
      • PubMed search introduces temporal bias toward published (completed) trials.
      • Deduplication uses fuzzy title matching (Levenshtein ≥ 0.85); false merges are possible. A validation sample of 50 merged and 50 near-miss unmerged pairs is provided in validation/ for independent scoring.

      Ethics statement

      This study uses only publicly available trial registration metadata from ClinicalTrials.gov and PubMed article metadata. No individual patient data, protected health information, or restricted-access datasets are used. Ethical approval was not required as no human subjects research was conducted.
